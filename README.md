# Restaurant Billing System (Java)

A console-based restaurant billing system developed using Java.  
This project demonstrates the use of core Java concepts and
Object-Oriented Programming principles.

## Features
- Menu-driven console interface
- Add food items with quantity
- Remove ordered items
- View current order list
- Automatic bill calculation
- GST calculation
- Final receipt generation

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Console I/O

## Project Structure
src/
└── restaurantbillingsystem/
├── Main.java
├── Menu.java
├── MenuItem.java
├── OrderItem.java
└── Bill.java

## How to Run
1. Clone the repository
2. Open the project in IntelliJ IDEA / Eclipse
3. Run `Main.java`
4. Follow on-screen menu instructions
